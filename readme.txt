== Description ==

O Chat App Brasil é um plugin que adiciona um botão do WhatsApp em diferentes páginas do seu site. Com este plugin, os usuários poderão entrar em contato com você facilmente através do WhatsApp.

Após a ativação do plugin, será adicionado um novo item de menu chamado "WhatsLp" no painel de administração.

Clique em "WhatsLp" para acessar as configurações do plugin.

Na página de configurações, você poderá personalizar o comportamento e a aparência do botão do WhatsApp.

Preencha os campos de configuração conforme suas preferências:

Páginas para ocultar o botão WhatsApp: Insira a URL completa das páginas em que você deseja ocultar o botão WhatsApp, uma por linha. Por exemplo: https://site.com.br/pagina-teste. Certifique-se de incluir a barra ("/") no final da URL.
Tamanho do botão WhatsApp: Defina o tamanho do botão WhatsApp. Utilize um valor em pixels, por exemplo: 80px.
Número de telefone do WhatsApp: Insira o número de telefone do WhatsApp, por exemplo: 5511999999999.
Posição do botão WhatsApp: Selecione a posição do botão WhatsApp na página.
Valor de z-index do botão WhatsApp: Defina o valor de z-index para o botão WhatsApp.
Clique no botão "Salvar alterações" para salvar as configurações.


== Installation ==

A maneira mais rápida de instalar o plugin é:

vá na página: Plugins -> Adicionar Novo
Pesquise por: "Chat App Brasil Plugin"
Clique em "Instalar"
Finaliza clicando em "Ativar Plugin & executar Chat App Brasil Plugin"

Se você preferir, você pode instalar de forma manual.

Suba a pasta "Chat-App-BR" para a pasta "/wp-content/plugins/"
Ative o plugin dentro do menu "Plugins" do WordPress.

Você encontrará o menu de "WhatsLP" no seu painel de administração do WordPress.

== Adicionando o botão WhatsApp ==

Após configurar o plugin, o botão do WhatsApp será adicionado automaticamente nas páginas selecionadas. O botão será exibido na posição escolhida e os usuários poderão clicar nele para iniciar uma conversa no WhatsApp.

Lembre-se de que o botão será ocultado nas páginas especificadas na configuração "Páginas para ocultar o botão WhatsApp".
